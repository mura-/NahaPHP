## classの継承
* 親クラスはしーじゃーと呼ぶ
* なおしーじゃクラスに定義されているものはオーバライド出来ない。
これをしーじゃ権力と呼ぶ。
* しーじゃクラスにはしばしばメモリリーク等をおこし迷惑をかけるものがある。
それらをやっけーしーじゃーと呼ぶ。
